CPU 性能测试#

CPU 性能直接影响服务器的运算能力，对于网站响应速度和并发处理能力都有重要影响。我使用 sysbench 工具对 DMIT 洛杉矶 TINY 套餐的 CPU 进行了测试。

平均延迟只有 0.24 毫秒，最小延迟 0.21 毫秒，即使是 95% 的任务也能在 0.27 毫秒内完成。这说明 CPU 的响应速度很快，不会成为系统的瓶颈。

IO 性能分析#

从磁盘 I/O 测试结果来看，这台 VPS 在随机读写方面表现稳定，延迟指标也有一定的参考价值。下面我们先从数据入手，分析它在不同维度的表现。

  • 随机读取： 1,986 IOPS / 7.8 MB/s - 中等
  • 随机写入： 1,983 IOPS / 7.8 MB/s - 中等
  • 读取延迟： 平均442.78微秒 - 一般
  • 写入延迟： 平均58.77微秒 - 良好

从这些数据可以看出，DMIT 的磁盘表现出典型的 SSD 特征：

  • 读写 IOPS 均衡：读写都接近 2000 IOPS，性能比较平衡，没有明显短板
  • 写入延迟优秀：58 微秒的写入延迟表现很好，说明写入缓存优化到位
  • 读取延迟偏高：442 微秒的读取延迟相对较高，可能受到虚拟化层面的影响

这种性能水平在入门级 VPS 中属于中等表现。虽然达不到高端 NVMe SSD 的水准，但对于大多数 Web 应用场景（如个人博客、企业官网、小型电商）来说，这样的 IO 性能完全能够满足日常使用需求。

DMIT 的主要优势在于网络质量，特别是到中国大陆的连接稳定性更好。虽然价格可能比一些竞品略高，但线路质量确实更优。

DMIT 跟 搬瓦工 的 CN2 线路质量差不多，但是价格更便宜；而 Vultr 价格更低，但是普通线路网络质量远远不如 DMIT。

Q6：流媒体解锁能力如何，能看 Netflix 吗？#

+

DMIT 的流媒体解锁能力一般，Netflix 只能看原创内容，Disney+ 被封禁。但是 YouTube Premium 和 Amazon Prime 可以正常使用。

如果你的主要需求是流媒体解锁，建议考虑专门的流媒体 VPS 或者解锁服务，例如 丽萨主机 提供家宽 IP，解锁能力更强。
